# Спецификация: Кастомизация UI сообщений в чат-оверлее

## Общая структура UI настроек

Настройки разделены на два таба:
1. **Фон**
2. **Контент**

---

## Таб 1: Фон

### 1.1 Шапка и подвал

| Настройка | Описание |
|-----------|----------|
| Изображение шапки | Загрузка/выбор изображения для верхней части |
| Изображение подвала | Загрузка/выбор изображения для нижней части |
| Смещение (translate) | Оффсет для точного позиционирования шапки/подвала |

> Отступы для шапки и подвала не задаются — только позиционирование через оффсеты (translate).
> Шапка и подвал привязаны к **Content-контейнеру** (не к фоновому блоку).

### 1.2 Тип фона (табы/переключатель)

Три варианта типа фона:
- **Цвет** — компонент уже готов
- **Градиент** — компонент уже готов
- **Изображение** — новый компонент

#### Настройки для типа "Изображение"

| Настройка | Тип | Описание |
|-----------|-----|----------|
| Цвет фона | color | Фоновый цвет под изображением |
| Цвет обводки | color | Цвет границы |
| Ширина обводки | number (px) | Толщина границы |
| Цвет тени | color | Цвет box-shadow |
| Радиус тени | number (px) | Размытие тени |
| Скругление углов | number (px) | border-radius |

### 1.3 Отступы

#### Внешние отступы (для всего контейнера с фонами)

| Настройка | Тип |
|-----------|-----|
| По горизонтали | number (px) |
| По вертикали | number (px) |

#### Внутренние отступы (для каждого слоя фона)

Задаются отдельно для каждого типа фона:
- Изображение
- Сплошной цвет
- Градиент

## Таб 2: Контент

### 2.1 Заголовок (Header)

#### Основные настройки

| Настройка | Тип | Варианты/Описание |
|-----------|-----|-------------------|
| Расположение заголовка | select | `top` (сверху), `left` (слева) |
| Фон заголовка | toggle | Включить/выключить фон |
| Кастомный цвет текста | toggle | Включить/выключить кастомный цвет |
| Цвет текста заголовка | color | Только при включённом кастомном цвете. Если выключено — используются цвета Twitch |
| Шрифт | font-picker | Выбор шрифта |
| Размер шрифта | number (px) | — |

#### Позиционирование заголовка

| Настройка | Тип | Варианты/Описание |
|-----------|-----|-------------------|
| Положение относительно контейнера | select | `inside` (внутри), `outside` (снаружи) |
| Выравнивание (для режима "сверху") | select | `left`, `center`, `right` |
| Оффсет позиционирования | translate (x, y) | Смещение заголовка |

#### Настройки фона заголовка

Фон заголовка выполняется только цветом (без изображения/градиента).

| Настройка | Тип |
|-----------|-----|
| Цвет | color |
| Цвет обводки | color |
| Ширина обводки | number (px) |
| Скругление углов | number (px) |
| Цвет тени | color |
| Радиус тени | number (px) |

### 2.2 Эмоуты в заголовке

#### Основные настройки

| Настройка | Тип | Варианты/Описание |
|-----------|-----|-------------------|
| Расположение рендера | select | `inside` (внутри контейнера заголовка), `outside` (снаружи контейнера заголовка) |
| Источник эмоутов | toggle | Кастомные (отдельная настройка) / Twitch |

#### Поведение `inside`

Эмоуты рендерятся как часть flex-контейнера заголовка (рядом с бейджами и ником). Отдельных настроек фона нет — эмоуты визуально являются частью заголовка.

#### Поведение `outside`

Эмоуты рендерятся снаружи контейнера заголовка. Фон наследуется от фона заголовка.

| Настройка | Тип | Варианты/Описание |
|-----------|-----|-------------------|
| Положение | select | `left` (слева от заголовка), `right` (справа) |
| Расстояние от заголовка | number (px) | gap между эмоутами и заголовком |

> Если заголовок в режиме `outside` (снаружи Content-контейнера), эмоуты `outside` позиционируются относительно заголовка и тоже находятся снаружи фона.

### 2.3 Контент (текст сообщения)

Контент — текст с инлайн HTML (эмодзи, эмоуты).

| Настройка | Тип | Описание |
|-----------|-----|----------|
| Цвет текста | color | — |
| Выравнивание | select | `left`, `center`, `right` |
| Шрифт | font-picker | — |
| Размер шрифта | number (px) | — |
| Цвет тени | color | text-shadow |
| Радиус тени | number (px) | размытие text-shadow |
| Размер эмоутов | number (px) | размер инлайн-эмоутов |

## Рендер и архитектура

### Текущая реализация (legacy)

| Файл | Назначение |
|------|------------|
| `ChatMessage.jsx` | Рендер сообщения, тянет данные из темы (legacy) |
| `MessageSettingsBlock.jsx` | Компонент настроек (только старые механики) |

Legacy-рендерер остаётся без изменений. Продолжает работать с `theme.chatMessage`.

### Подход к реализации v2

- **Без миграции** — не трогаем старую структуру темы
- **Новый объект `v2`** в теме для хранения стилизации нового рендера
- **Новые компоненты с нуля** — v2-рендерер пишется отдельно, legacy не модифицируется
- Новый рендерер читает из `theme.v2`, старый продолжает работать с `theme.*` (legacy)
- Выбор рендерера: если `theme.v2` существует — используется v2, иначе legacy

### DOM-структура v2

**Принцип:** Content-контейнер является фоновым блоком (фоны применяются к нему напрямую). Шапка и подвал (декоративные изображения) всегда внутри Content.

#### Header inside (внутри Content)

```html
<Wrapper>                           <!-- margin, общий контейнер -->
  <Content>                         <!-- фон: цвет/градиент/изображение, padding -->
    <HeaderDecor />                 <!-- декоративная картинка-шапка, translate -->
    <Header>                        <!-- заголовок внутри контента -->
      <Emotes />                    <!-- если inside — часть flex-контейнера -->
      <Badges /> <Name />
    </Header>
    <Text />                        <!-- текст сообщения -->
    <FooterDecor />                 <!-- декоративная картинка-подвал, translate -->
  </Content>
</Wrapper>
```

#### Header outside, layout top (снаружи, сверху)

```html
<Wrapper>
  <Header>                          <!-- заголовок снаружи, над Content -->
    <Emotes />                      <!-- если outside — рядом с header -->
    <Badges /> <Name />
  </Header>
  <Content>                         <!-- фон: цвет/градиент/изображение -->
    <HeaderDecor />
    <Text />
    <FooterDecor />
  </Content>
</Wrapper>
```

#### Header outside, layout left (снаружи, слева)

```html
<Wrapper style="flex-direction: row">
  <Header>                          <!-- заголовок снаружи, слева -->
    <Emotes />
    <Badges /> <Name />
  </Header>
  <Content>
    <HeaderDecor />
    <Text />
    <FooterDecor />
  </Content>
</Wrapper>
```

### Фоновые слои на Content

Content-контейнер поддерживает три слоя фона с разными padding:
- **Цвет** — базовый слой (z-index 0)
- **Градиент** — поверх цвета (z-index 1)
- **Изображение** — верхний слой (z-index 2)

Каждый слой может иметь свой padding (inset), реализуется через вложенные `::before`/`::after` или абсолютно-позиционированные div'ы внутри Content.

### Структура данных (финальная)

> Все числовые значения — `number` (не строки). Цвета — hex-строки. Opacity — `number` 0–1.

```js
theme: {
  // ... legacy поля (не трогаем)

  v2: {
    message: {
      // ═══════════════════════════════════════
      // Таб 1: Фон
      // ═══════════════════════════════════════
      background: {
        // Декоративные изображения (привязаны к Content)
        headerDecor: {
          image: null,                    // string (URL) | null
          translate: { x: 0, y: 0 }
        },
        footerDecor: {
          image: null,                    // string (URL) | null
          translate: { x: 0, y: 0 }
        },

        // Активный тип фона (один из трёх)
        type: 'color',                    // 'color' | 'gradient' | 'image'

        // Настройки цветного фона
        color: {
          color: '#422434',
          opacity: 0.77,
          borderColor: '#0af0d5',
          borderOpacity: 1,
          borderRadius: 12,
          shadowColor: '#ffffff',
          shadowOpacity: 1,
          shadowRadius: 5
        },

        // Настройки градиентного фона
        // (совместим с legacy-форматом GradientEditor)
        gradient: {
          gradients: [{
            type: 'linear',               // 'linear' | 'radial'
            angle: 90,
            center: { x: 50, y: 50 },    // для radial
            stops: [
              { id: 'uuid', color: '#9b74ff', alpha: 1, position: 0 },
              { id: 'uuid', color: '#7c4dff', alpha: 1, position: 100 }
            ]
          }]
        },

        // Настройки фона-изображения
        image: {
          src: null,                      // string (URL) | null
          backgroundColor: '#000000',
          borderColor: '#ffffff',
          borderWidth: 0,
          borderRadius: 0,
          shadowColor: '#000000',
          shadowRadius: 0
        },

        // Внешние отступы (Wrapper)
        margin: { horizontal: 0, vertical: 0 },

        // Внутренние отступы Content (общий padding контента)
        padding: { top: 0, right: 0, bottom: 0, left: 0 },

        // Inset каждого фонового слоя (смещение от краёв Content)
        layerInset: {
          color:    { top: 0, right: 0, bottom: 0, left: 0 },
          gradient: { top: 0, right: 0, bottom: 0, left: 0 },
          image:    { top: 0, right: 0, bottom: 0, left: 0 }
        }
      },

      // ═══════════════════════════════════════
      // Таб 2: Контент
      // ═══════════════════════════════════════
      content: {
        header: {
          layout: 'top',                  // 'top' | 'left'
          position: 'inside',             // 'inside' | 'outside'
          align: 'left',                  // 'left' | 'center' | 'right'
          translate: { x: 0, y: 0 },

          font: {
            family: 'Arial',
            url: null,                    // string (URL) | null — для кастомных шрифтов
            size: 14
          },

          customColor: {
            enabled: false,               // false = цвет Twitch
            color: '#ffffff'
          },

          background: {
            enabled: false,
            color: '#000000',
            opacity: 0.5,
            borderColor: '#ffffff',
            borderOpacity: 1,
            borderWidth: 0,
            borderRadius: 8,
            shadowColor: '#000000',
            shadowOpacity: 0,
            shadowRadius: 0
          },

          emotes: {
            position: 'inside',           // 'inside' | 'outside'
            source: 'twitch',             // 'twitch' | 'custom'
            // Только для position: 'outside':
            placement: 'right',           // 'left' | 'right'
            gap: 4
          }
        },

        text: {
          color: '#ffffff',
          opacity: 1,
          align: 'left',                  // 'left' | 'center' | 'right'
          font: {
            family: 'Arial',
            url: null,
            size: 14
          },
          shadowColor: '#000000',
          shadowOpacity: 0,
          shadowRadius: 0,
          emoteSize: 24
        }
      }
    }
  }
}
```
