# Аудиовизуализаторы — параметры URL

Все три аудиовизуализатора поддерживают настройку через query-параметры URL.
Если параметр не указан, используется значение по умолчанию.

Базовый URL: `http://localhost:5173` (dev).

---

## Linear FFT (`/audio-fft-linear-demo`)

Горизонтальный столбчатый спектр.

| Параметр           | Тип     | По умолчанию         | Описание                           |
|--------------------|---------|----------------------|------------------------------------|
| `width`            | string  | `1280px`             | Ширина контейнера (CSS-значение)   |
| `height`           | string  | `350px`              | Высота контейнера (CSS-значение)   |
| `bars`             | int     | `256`                | Количество частотных полос         |
| `barColor`         | string  | `#37ff00`            | Цвет столбцов                      |
| `barGradient`      | boolean | `true`               | Включить градиентную заливку       |
| `backgroundColor`  | string  | `rgba(0,0,0,0.00)`  | Цвет фона канваса                  |
| `smoothDuration`   | float   | `16`                 | Длительность сглаживания (мс)      |
| `amplitude`        | float   | `1`                  | Масштаб по вертикали               |
| `peakHold`         | float   | `10`                 | Время удержания пика (мс)         |
| `peakFall`         | float   | `800`                | Длительность падения пика (мс)     |
| `peakColor`        | string  | `#ce00ff`            | Цвет индикатора пика               |
| `peakThickness`    | float   | `2`                  | Толщина линии пика (px)            |

### Примеры

По умолчанию (зелёные столбцы, фиолетовые пики):
```
/audio-fft-linear-demo
```

Широкий оверлей, бирюзовые столбцы, жёлтые пики, 128 полос:
```
/audio-fft-linear-demo?width=1920px&height=200px&bars=128&barColor=%2300e5ff&peakColor=%23ffeb3b
```

Высокий детальный визуализатор с медленным сглаживанием:
```
/audio-fft-linear-demo?width=800px&height=600px&bars=512&smoothDuration=32&amplitude=1.5
```

Без пиков, красные столбцы:
```
/audio-fft-linear-demo?barColor=%23ff0000&peakThickness=0
```

## Round FFT (`/audio-fft-round-demo`)

Круговой (кольцевой) спектр.

| Параметр            | Тип     | По умолчанию         | Описание                                      |
|---------------------|---------|----------------------|-----------------------------------------------|
| `width`             | string  | `500px`              | Ширина контейнера (CSS-значение)              |
| `height`            | string  | `500px`              | Высота контейнера (CSS-значение)              |
| `bars`              | int     | `256`                | Количество частотных полос                    |
| `barColor`          | string  | `#ce00ff`            | Цвет секторов                                 |
| `barGradient`       | boolean | `true`               | Включить градиентную заливку                  |
| `backgroundColor`   | string  | `rgba(0,0,0,0.00)`  | Цвет фона канваса                             |
| `smoothDuration`    | float   | `60`                 | Длительность сглаживания (мс)                 |
| `peakHold`          | float   | `500`                | Время удержания пика (мс)                    |
| `peakFall`          | float   | `800`                | Длительность падения пика (мс)                |
| `peakColor`         | string  | `#57fe04`            | Цвет дуги пика                                |
| `peakThickness`     | float   | `2`                  | Толщина дуги пика (px)                        |
| `innerRadiusRatio`  | float   | `0.6`                | Внутренний радиус как доля от внешнего (0–1)  |
| `sectorGap`         | float   | `0.01`               | Зазор между секторами (радианы)               |
| `startAngle`        | float   | `-1.5708` (−π/2)    | Начальный угол (радианы, верх = −π/2)         |
| `inactiveSector`    | float   | `0`                  | Размер неактивной дуги (радианы)              |

### Примеры

По умолчанию (фиолетовое кольцо, зелёные пики):
```
/audio-fft-round-demo
```

Большое кольцо, бирюзовый цвет, тонкое внутреннее отверстие:
```
/audio-fft-round-demo?width=800px&height=800px&barColor=%2300e5ff&innerRadiusRatio=0.3
```

Полукруг (нижняя половина неактивна):
```
/audio-fft-round-demo?inactiveSector=3.14159&startAngle=-3.14159
```

64 полосы с широкими зазорами, без градиента:
```
/audio-fft-round-demo?bars=64&sectorGap=0.05&barGradient=false&barColor=%23ff6600
```

## Waveform (`/audio-waveform-demo`)

Визуализатор в стиле осциллографа.

| Параметр           | Тип     | По умолчанию            | Описание                        |
|--------------------|---------|-------------------------|---------------------------------|
| `width`            | string  | `1280px`                | Ширина контейнера (CSS-значение)|
| `height`           | string  | `350px`                 | Высота контейнера (CSS-значение)|
| `lineColor`        | string  | `#37ff00`               | Цвет линии                      |
| `lineWidth`        | float   | `2`                     | Толщина линии (px)              |
| `backgroundColor`  | string  | `rgba(197,89,89,0.06)`  | Цвет фона канваса              |
| `gridColor`        | string  | `rgba(255,255,255,0.1)` | Цвет сетки                      |
| `centerLineColor`  | string  | `rgba(255,255,255,0.3)` | Цвет центральной линии          |
| `showGrid`         | boolean | `true`                  | Показывать фоновую сетку        |
| `showCenterLine`   | boolean | `true`                  | Показывать центральную линию    |
| `smoothDuration`   | float   | `16`                    | Длительность сглаживания (мс)   |
| `amplitude`        | float   | `1`                     | Масштаб амплитуды               |

### Примеры

По умолчанию (зелёная линия, сетка включена):
```
/audio-waveform-demo
```

Чистая волна без сетки, белая линия:
```
/audio-waveform-demo?lineColor=%23ffffff&showGrid=false&showCenterLine=false&backgroundColor=rgba(0,0,0,0)
```

Широкий оверлей, толстая красная линия, усиленная амплитуда:
```
/audio-waveform-demo?width=1920px&height=150px&lineColor=%23ff0000&lineWidth=4&amplitude=2
```

## Заметки

- **Цвета**: символ `#` нужно кодировать как `%23` (например `%2300e5ff` для `#00e5ff`). Значения `rgba(...)` работают как есть.
- **Булевы значения**: принимаются `true`/`1` и `false`/`0`.
- **Углы** (Round FFT): значения в радианах. Частые значения: `0` = вправо, `1.5708` (π/2) = вниз, `-1.5708` (−π/2) = вверх, `3.14159` (π) = влево.
- Все визуализаторы по умолчанию подключаются к `ws://localhost:5001` для получения FFT-данных.
